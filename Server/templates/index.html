Welcome to demo fintech app

This application uses uses PSD2 based access to commercial banks and implements (Sandbox mode) both major PSD interfaces - Account Information Services (AISP) and Payment Initiation Services (PISP).

Flow for each bank consists (or should consist, based on API completeness) on following steps:

  • Authentication - secure two factor authentication to bank of customers choosing
  • Consent - customer's permission to use particular services
  • AISP - get list of accounts (checking accounts only)
  • AISP - get information on particular account
  • AISP - get transaction list for particular account
  • PISP - initiate payment on customer's behalf
